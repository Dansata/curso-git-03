Repositorios remotos

Crear Repositorio vacío en github

Primero creo un repositorio vacio en git hub

$ git add remote

Una vez creamos el repositorio en GitHub, copiamos la dirección y sincronizamos con el repositorio local. Para esto, usamos el comando $ git remote add origin https://github.com/User/nombreRepositorio.git

Una vez sincronizado, subimos la rama con nuestro repositorio con el comando $ git push origin master

La terminal nos pedirá usuario y contraseña de Github y luego lo subirá
